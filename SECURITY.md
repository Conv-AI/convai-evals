# Security Policy

Do not include customer datasets, credentials, endpoint secrets, or private infrastructure identifiers in issues, pull requests, examples, or reports.

To report a vulnerability, contact Convai through the security process listed on the main Convai website or use the private disclosure channel configured for the GitHub organization.

Public examples in this repo must remain synthetic and customer-neutral. Reports may contain backend IDs; treat generated reports as operational artifacts and share them only with the appropriate audience.
